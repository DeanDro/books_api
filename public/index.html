Book Reviews API

Documentation

URL Sample

Sample url starting at:
localhost:3000/review/book_name

If more than one books have the same name, then more than one books will be returned in the json format. This will be included under the "books" key.

Description

This API returns reviews from over 11,000 books given by Goodreads. The API contains information about:

  • bookID: This key contains the id number for that particular book in the database. It is an integer value in a string format and is the first value in the json.

  • title: This is a string value that contains the title of the book. If an incomplete title is given and more than one books match the value, the API will return all possible values. For example, if the user gives as title "Harry Potter", then the API will return all 7 Harry Potter books.

  • authors: This is a string value that contains the name of the author or authors that wrote that particular book.

  • average_rating: This is a float number that comes in a string format and contains the average rating given for that particular book.

  • isbn: This is an integer value that comes in a string format and contains the isbn number for that particular book.

  • isbn13: Similarly, this is an integer value that comes in a string format and contains the isbn13 number for that particular book.

  • language_code: This is a string value that contains the code of the language the book was written. For example "eng" stand for English.

  • num_pages: Returns the number of pages in that particular book. The format is a string of an integer number.

  • ratings_count: It returns an integer in a string format with the number of ratings given to that book.

  • text_reviews_count: This integer in the format of a string gives the number of times the book has been reviewed.
